# HX-AM (Hybrid-X Anomaly Miner) — Thin AI Proxy

**Version:** 3.4 | **Status:** MVP | **License:** MIT

## О проекте

HX-AM — это прокси-сервер между пользователем и LLM (Groq/llama-3.3-70b), который анализирует запросы на наличие структурных изоморфизмов между разными масштабами и научными доменами. Система генерирует проверяемые гипотезы и сохраняет инсайты в виде артефактов.

Ключевая метрика — **B_sync** (0.00–1.00), основанная на экспоненциальном затухании и структурном сходстве. Значение >0.50 инициирует создание артефакта.

## Быстрый старт

1. Клонируйте репозиторий  
2. Установите зависимости: `pip install fastapi uvicorn litellm python-dotenv`  
3. Создайте `.env` с `GROQ_API_KEY`  
4. Запустите: `python proxy_server.py`  
5. Откройте `http://localhost:8000`

## Структура

- `proxy_server.py` — FastAPI сервер, эндпоинты  
- `chat_history_class.py` — логирование и статистика  
- `system_prompt.txt` — инструкции для LLM (можно редактировать)  
- `index.html` — веб-интерфейс (3 вкладки)  
- `artifacts/` — сохранённые JSON-артефакты (не в репозитории)  
- `chat_history/` — JSONL-логи (не в репозитории)

## Ограничения MVP

- Rate limit Groq (12k токенов/мин)  
- Нет авторизации, очередей, контейнеризации
